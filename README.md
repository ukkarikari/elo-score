# elo-score
general use elo scoreboard in react native

### deployment instructions

clone the repo
```git clone https://github.com/ukkarikari/elo-score/ ```

go into folder

```cd elo-score/```

install dependencies

```npm install```

run server in expo go

```npm run start```
